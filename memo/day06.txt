1. 제어문(Control Statement)
	제어문은 프로그램의 실행 흐름을 제어하기 위해 사용한다
	대표적으로 조건문, 반복문, 기타 제어문이 있다

1) 조건문 : 조건에 따라(조건의 결과는 항상 boolean 타입) 실행할 문장을 선택할 때 사용한다
- if문	
(1) if문 : 조건식의 결과가 true일 때만 실행

	if(조건식){
		조건식이 참일 때 실행할 문장;
	}

(2) if~else문 : 조건식의 결과가 true(참)/false(거짓) 두가지 경우 일 때 사용

	if(조건식){
		조건식의 결과가 참일 때 실행될 문장;
	}else{
		조건식의 결과가 거짓일 때 실행될 문장;
	}

(3) if~else if~else문 : 조건이 2개 이상일 때 사용
			위에서부터 조건을 검사하며 true가 제일 먼저 나오는 하나만 실행

	if(조건식1){
		조건식1의 결과가 참일 때 실행될 문장;
	}else if(조건식2){
		조건식1의 결과가 거짓이고 조건식2의 결과가 참일 때 실행될 문장;
	}else if(조건식3){
		조건식1과 조건식2의 결과가 거짓이고 조건식3의 결과가 참일 때 실행될 문장;
	}else{
		조건식1과 조건식2, 조건식3의 결과가 모두 거짓일 때 실행될 문장;
	}

- switch문 : 값을 비교하여 분기 처리
	경우의 수가 명확할 때 주로 사용(ex : 계절-봄,여름,가을,겨울 / 짝수, 홀수 등)

	switch(변수){
	case 값1:
		실행문;
		break; 	//case마다 break문을 반드시 써야한다!
	case 값2:
		실행문;
		break;
	...
	default: 	//모든 case에 해당하지 않을 때 실행
		실행문;
		break;
	}

2) 반복문 : 같은 코드를 여러번 반복해서 실행하고 싶을 때 사용

- for문 : 몇 번 반복할 지 알 때 주로 사용(범위/반복횟수가 정해져있을때)
	for(초기식; 조건식; 증감식){
		실행문;
	}

	실행순서
	a. 초기식 int i = 0;
	b. 조건식 검사(결과가 true라면 중괄호 영역으로 들어가고 false라면 중괄호 영역 밖으로 빠져나온다)
	c. 실행문(조건식 결과가 true일때)
	d. 증감식
	e. 조건식을 다시 확인하고 b부터 e까지 반복 진행(조건식이 false가 될 때까지 반복)

- while문 : 조건식이 ture인 동안 반복(반복 횟수가 정해져 있지 않을 때 사용)
	조건식이 항상 true면 무한반복(무한루프) 발생하므로 조건식의 결과가 false가 되는 경우를 만들어야한다
	
	while(조건식) {
		실행문;
	}

- do~while문 : 무조건 한 번은 실행
	실행 후에 조건 검사(조건 검사시 결과가 true면 다시 중괄호 안의 영역실행, false면 중괄호 영역 벗어남)
	
	do {
		실행문;
	}while(조건식);
	
3) 기타제어문 : 반복문의 실행 흐름을 강제로 제어할 때 사용한다
- break문 : 반복문(for, while, do~while) 또는 switch문을 즉시 종료할 때 사용한다
	break가 실행되는 순간 반복문 전체가 종료되며
	break 이후의 코드는 실행되지 않는다

- continue문 : 반복문에서 현재 반복을 건너 뛰고 다음 반복으로 바로 이동할 때 사용한다
	반복문은 종료되지 않으며 continue 아래에 있는 코드는 실행되지 않는다

2. 연산자
1) 연산자의 우선순위
	최우선 연산자	가장 먼저 계산됨
			() 괄호 	. 객체 접근  [] 배열접근
	단항 연산자	피연산자가 1개
			++ --
			!
			+ -(부호 양수와 음수를 의미)
	산술 연산자	수학계산
			+(덧셈) -(뺄셈) *(곱셈) /(나머지 몫자체) %(나머지 연산자)
	쉬프트 연산자	비트이동 연산자(보통 2진수로 계산 시 사용)
			<< 왼쪽으로 이동
			>> 오른쪽으로 이동
	관계 연산자	크기 비교시 사용되며 조건식으로 많이 사용됨★
			관계연산자의 결과는 boolean 타입
			> < >= <= == !=
	논리 연산자	조건연결 시 사용
			&& (AND) || (OR) ! (NOT)
	삼항 연산자	피연산자가 3개
			조건에 따라 값을 선택할 때 사용(간단한 조건식일 경우)
			조건식 ? 참일 때의 값 : 거짓일 때의 값;

	대입 연산자	값 저장 시 사용 
			가장 마지막에 실행됨
			= += -= *= /=
