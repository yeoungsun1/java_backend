2) 입력메소드

	sc.nextInt();	입력받은 값을 정수형(int) 타입으로 돌려준다
			엔터를 임시 저장공간에서 소모하지 않는다
	sc.nextDouble();	입력받은 값을 실수형(double) 타입으로 돌려준다
			엔터를 임시 저장공간에서 소모하지 않는다

1. 클래스와 객체
	클래스 : 사용자 정보를 저장하기 위한 틀(설계도면)
	필드 : 이름, 나이, 아이디, 비밀번호
	메소드 : 출력하는 기능, 로그인 기능
	생성자 : 객체를 만들 때 초기값 설정 가능

	객체 : 설계도면에서 나온 실체(메모리에 올라간 것)

+) 클래스(class)
	공통된 속성과 기능을 하나로 묶어 놓은 틀(설계도)

- 필드	각각의 속성(객체마다 속성의 값은 다르다, 같을 수도 있다)
- 생성자	클래스로 객체를 만들 때 자동으로 만들어진다(기본생성자, 컴파일러가 만든다)
	매개변수가 있는 생성자를 우리가 직접 만들게 되면 객체를 만들면서 초기값을 넣어줄수있다
	단, 우리가 직접 만들게 되면 기본 생성자는 만들어지지 않는다
- 메소드	하나의 기능(행동, 행위, 동작)
- 객체	클래스로 만들어낸 실체( ex: new Scanner(System.in);
				new User();)

2. 메소드 정의와 호출
1) 함수와 메소드
	둘 다 기능을 의미한다
	클래스 밖에 선언되어있는지, 클래스 안에 선언되어있는지에 따라서 함수/메소드 라는 용어로
	부른다.
	자바는 무조건 클래스 안에서 코드가 작성되기 때문에 클래스 밖에 선언할 수 없다.
	따라서 자바에서는 모든 함수를 메소드라고 부른다.
	객체명.메소드로 호출(메소드를 사용한다)해야한다

2) 메소드
	이름뒤에 소괄호
	단, 키워드 뒤에 소괄호는 메소드가 아니다

3) 메소드 선언과 구현(정의)
	리턴타입 메소드명(자료형 매개변수, ...)	//선언부
	{				//구현부
		실행할 문장;
		return 리턴값;
	}

리턴타입 : 반환할 값이 있다면 반환값(리턴값)의 자료형을 작성한다
	없다면 void를 작성한다
메소드명 : 동사로 작성한다. 카멜 표기법을 사용한다
매개변수 : 외부에서(사용할 곳에서) 전달받을 값이 있다면, 자료형과 순서에 맞게 선언해준다
	(★ 반드시 자료형이 필요하다)
	전달 받을 값이 없다면 생략이 가능하다(단, 생략시 외부에서 값을 받을 수 없다)
실행할 문장 : 생략이 가능하지만 생략하면 기능을 할 수 없다.
	메소드의 기능을 구현하는 코드를 작성한다
return : 생략이 가능하며 return이 실행되면 메소드가 종료된다.
	리턴값이 있다면 메소드를 사용한 부분을 통채로 리턴 "값"으로 본다.
	리턴값이 없다면 값이 아니다.

4) 메소드 호출 방법 4가지
	매개변수	리턴값	호출방법
	X	X	객체명.메소드명();
	O	X	객체명.메소드명(인수);
	X	O	자료형 변수명 = 객체명.메소드명();
			syso(객체명.메소드명());
	O	O	자료형 변수명 = 객체명.메소드명(인수);
			syso(객체명.메소드명(인수));


+) 숫자 2개를 더해서 출력하는 기능을 하는 메소드
	반환타입 메소드명(자료형 매개변수){
		실행할문장;
		return 리턴값;
	}

1) 기능을 생각한다(숫자2개를 더해주는 기능)
2) 반환타입을 생각한다. 반환타입이 생각나지 않는다면 우선 void로 작성한다
	void
3) 기능에 알맞은 메소드명을 작성한다(가능하면 동사로 add, sum, ..)
	void add(){}
4) 매개변수를 생각한다(매개변수2개가 필요하다, 정수형으로 타입 선언)
	void add(int num1, int num2){}
5) 실행할 문장을 작성한다
	void add(int num1, int num2){
		int result = num1 + num2;
		syso(result);
	}
6) 리턴값을 생각한다
(return을 하지 않은 경우라면 void, return을 사용하는 경우라면 리턴값의 타입으로 작성한다)
	int add(int num1, int num2){
		int result = num1 + num2;
		syso(result);
		return result;
	}


=> add 메소드를 호출하는 경우 매개변수가 있고, 리턴값이 있으므로 출력문(객체명.메소드명(정수형값, 정수형값));
	method.add(10, 20);	=> 매개변수 o, 리턴값 x 호출
				결과는 30 출력
	syso(method.add(1, 2));	=> 매개변수 o, 리턴값 o 호출
				결과는 	3
					3
				출력된다
