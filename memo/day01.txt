3. 프로그램의 동작 원리
	<일반프로그램>		<자바프로그램>
	프로그램			프로그램
					JVM
	OS(운영체제)		OS(운영체제)
	HW			HW
	
- 일반 프로그램의 경우
	프로그램 실행 -> 프로그램을 구성하는 명령어와 데이터가 운영체제(os)에 의해 
	메모리(RAM)에 적재 -> 프로그램이 메모리에 적재되어 실행중인 상태(프로세스)
	-> CPU가 메모리에 있는 .exe파일을 해석 및 실행

- java 프로그램의 경우
	프로그램 실행 -> JVM이 프로그램과 운영체제 사이에서 존재 -> JVM이 운영체제와 
	소통하여 일반 프로그램과 달리 운영체제에 의존적이지 않음(이식성이 높다)

4. JVM, JRE, JDK
1) JVM(Java Virtual Machine)
	운영체제에 맞춰서 java로 만든 프로그램을 실행시켜준다
	.java 파일 -> 컴파일 -> .class 실행
	
2) JRE(Java Runtime Enviroment)
	JVM을 실행한다
	JVM을 실행할 때 필요한 라이브러리 파일 등을 가지고 있다

3) JDK(Java Devlopment Kit)
	JRE를 포함한다
	JRE 외에 개발에 필요한 도구들을 가지고 있다

5. JDK 설치
1) JDK 설치 확인 방법
	windows + r로 실행창을 열고 cmd 입력
	> javac -version 	컴파일러 버전 확인
	> java -version		설치한 자바 버전

2) JDK 설치 방법
	구글 검색창에 JDK17 검색 > 제일 최상단 오라클 링크 클릭
	> 아래쪽으로 내려서 Windows X64 Installer exe 파일 다운로드
	> 설치 프로그램 내 폴더\java\resource 이동
	> 더블클릭 > 실행 > 설치 중 설치 경로에 띄어쓰기가 있거나 한글이 있으면 오류가 나기 때문에
	> C:\Java\jdk-17로 경로 설정
	> 설치 진행 후 close 클릭

3) 메모장으로 java 프로그램 실행하기
(1) 메모장으로 JAVA 코드 작성하기
	- 돋보기 memo 검색 > 텍스트 문서 열기
	- 파일 > 다른 이름으로 저장 > 파일 이름을 Test.java 파일 형식은 모든 파일로 변경
	- 메모장에서 아래 코드 입력
public class Test {
   public static void main(String[] args) {
      System.out.println("hello java");
   }
}
	- 저장(ctrl + s)

(2) cmd에서 컴파일 & 실행
	- windows + r > cmd 입력
	- 파일이 있는 경로로 이동 
		드라이브 이동시 	>D:
		경로만 이동시 	>cd 경로
		C:>D:
		D:>cd D:\web_1900_kys\java\workspace

	- 컴파일
	D:\web_1900_kys\java\workspace> javac Test.java
		결과 아무것도 안나오고 해당 폴더로 이동하면 Test.class 파일이 생성됨
	- 실행
	D:\web_1900_kys\java\workspace> java Test
	hello java

6. 통합 개발환경(IDE) 설치
	IDE(Integrated Development Environment)
	다양한 언어를 지원하는 프로그래밍 통합 개발 환경이자 프로그래밍 도구

1) 이클립스 설치
	https://www.eclipse.org/downloads/packages/release	
	2024.12 클릭
	R Packages > Eclipse IDE for Enterprise Java and Developers (톱니바퀴)
	> Windows X86_64 (zip파일로 다운로드)
	
	설치 압축 풀기(반디집) > 폴더 내 eclipse.exe 마우스 우클릭 > 바로가기 만들기
	"바로가기 실행 파일"을 java 폴더로(memo, resource, workspace) 이동
	
2) 이클립스 설정
	window > preferences
(1) 이클립스 글씨체 변경(consolors 되어있는지 확인)
	font 검색 > Colors and Fonts 클릭 > Basic > Text Font 클릭하고 Edit 클릭
	> 원하는 글꼴과 크기 설정하고 확인 > Apply

(2) 인코딩 타입 확인
	enc 검색 > Workspace 클릭 > Text file encoding Other 확인 
	> UTF-8 확인(한글안깨짐) > Apply

(3) 컴파일러 변경
	compiler 검색 > level 21에서 17로 변경(jdk버전에 맞춰서 변경) > Apply

(4) 관점(Perspective) 변경
	perspec 검색 > General > Perspectives > java 클릭 후 Mack Default 클릭 > Apply
	> Apply and Close
