3) 프로그램 동작 원리
	[일반프로그램]
	프로그램을 실행 -> OS가 프로그램을 메모리(RAM)에 적재 
	-> CPU가 메모리에 있는 실행파일(.exe)을 해석 및 실행
	
	[JAVA 프로그램]
	프로그램을 실행 -> JVM이 프로그램과 OS 사이에서 중간 역할 -> 운영체제에 덜 의존(이식성이 높다)

4) JVM / JRE / JDK
	JVM(Java Virtual Machine)
		자바 프로그램 실행 담당
		.java -> 컴파일 -> .class 실행
	JRE(Java Runtime Environment)
		JVM + 실행에 필요한 라이브러리 포함
	JDK(Java Development Kit)
		JRE 포함
		개발에 필요한 도구(컴파일러 등) 포함
	=> 개발자는 JDK 설치 필수!!

5) JDK 설치 
	JDK 설치 확인
	CMD(명령 프롬프트) 
		> javac -version //컴파일러 버전
		> java -version // 자바 실행 버전
 
6) IDE(통합 개발 환경)
	코드 작성 + 실행 + 디버깅을 한 번에 할 수 있는 도구
	대표적인 Java IDE : Eclipse

1. 이클립스 단축키
1) 자바의 구성요소
	자바 프로젝트 > 패키지 > 클래스 > 메소드
	* 클래스의 첫글자는 대문자로 작성!!
	* 메소드의 이름 뒤에는 소괄호가 온다!!

2) 이클립스 단축키(Windows)
- 자바 프로젝트 생성, 패키지 생성, 클래스 생성
	ctrl + n > java p > 프로젝트 이름 > enter
	ctrl + n > pac > 패키지명 > enter
	ctrl + n > class > 클래스명(첫글자는 반드시 대문자!!!) > enter

- 메소드 생성(main 메소드)
	클래스 중괄호 영역 안에서 main이라고 쓰고 ctrl + space bar > enter
	public static void main(String[] args){}

	클래스생성 화면에서 alt + v 누르면 자동 생성된다

- 출력 메소드
	main 메소드 중괄호 영역 안에서 syso라고 쓰고 ctrl + space bar > enter
	System.out.println();	

- 저장
	ctrl + s

- 실행
	ctrl + F11

- 자동 들여쓰기(자동 정렬)
	ctrl + shift + f

- 한 줄 복사
	ctrl + alt + 위아래 방향키

- 한 줄 이동
	alt + 위아래 방향키

3) 기본 용어 정리
세미콜론 ;

3. 출력메소드
	System.out.println();	소괄호 안의 내용(값)을 콘솔창에 출력하고 줄바꿈이 된다
	System.out.print();	소괄호 안의 내용(값)을 콘솔창에 출력하고 줄바꿈이 되지 않는다
	System.out.printf();	형식지정자(서식문자)를 이용하여 콘솔창에 출력하고 줄바꿈이 되지 않는다

- 형식지정자의 종류

	문자	%c
	정수	%d
	실수	%f
	논리형	%s(문자열과 동일)

4. 변수(Variable)
	값을 저장하기 위한 저장공간
	=> 재사용하기 위해서 사용한다

1) 변수의 선언과 사용
	자료형 변수명;
	변수명 = 값;
	 = 대입연산자 
		대입연산자를 기준으로 오른쪽은 값, 왼쪽은 저장공간
		오른쪽에 있는 값을 왼쪽의 저장공간에 저장해줘~

	int age;	정수형 age 라는 변수를 선언했다
	age = 20;	age 저장공간에 20이라는 정수형 값을 저장했다
	
2) 변수의 선언과 동시에 초기화
	자료형 변수명 = 값;

3) 변수의 사용
	syso(변수명);	//해당 변수에 저장된 값이 출력된다

+) 자료형(DataType)
		타입	크기	범위		예시
	정수형	byte	1byte 	-128 ~ 127
		short	2byte	약 3만
		int	4byte	약 21억		10, 0, -35, -255, ..
		long	8byte			10L, -253L, 0L, ..
	실수형	double	8byte	소수점 15		10.52, -50.01111, 0.0, ...
		float	4byte	소수점 6~7	3.14f, -3.14F, 0.0f, ...
	논리형	boolean 1byte			true, false
	문자형	char	2byte			' '로 묶어진 1개의 문자
	문자열	String  ?			""로 묶어진 0개 이상의 문자
