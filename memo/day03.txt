0. 복습
1)출력메소드
   System.out.println();   소괄호 안의 값을 콘솔창에 출력하고 줄바꿈이 된다
   System.out.print();   소괄호 안의 값을 콘솔창에 출력하고 줄바꿈이 되지 않는다
   System.out.printf();   형식지정자를 이용하여 콘솔창에 출력하고 줄바꿈이 되지 않는다

   형식지정자의 종류
   문자열   %s   System.out.printf("%s", "문자열");
   논리형   %s   System.out.printf("%s %s", "문자열", true);
   문자형   %c   System.out.printf("%c", 'a');
   정수형   %d   System.out.printf("%d", "10");
   실수형   %f   System.out.printf("%f", "3.14");

2) 변수와 자료형
- 기본 자료형(Primitive)   값 자체 저장, Stack 메모리에 저장
   정수형   byte   1바이트(8비트)   -128 ~ 127
      short   2바이트(16비트)   -32768 ~ 32767
      int   4바이트(32비트)   -2,147,483,548 ~ 2,147,483,547
      long   8바이트(64비트)   
   실수형   double   8바이트      소수점 이하 15자리까지 표현 가능   실수형의 기본 타입
      float   4바이트      소수점 이하 6자리까지 표현 가능 (0.0f)
   논리형   boolean   1바이트      true, false
   문자형   char   2바이트      ' ' 묶어진 문자 1개

- 참조자료형(Reference)	직접 값을 저장하는 대신, 객체의 주소(위치)를 저장, Heap 메모리
	문자열	String		"" 묶어진 0개이상의 문자
	배열	int[], double[]		하나의 자료형으로 여러개의 값을 저장하는 저장공간의 나열
	클래스	User, Scanner, ArrayList

변수(Variable)
	값을 저장하기 위한 저장공간
	★ 하나의 자료형으로 하나의 값만 저장한다

	변수명 : 저장공간의 이름

	자료형 변수명;
	변수명 = 값;

	자료형 변수명 = 값; //변수의 선언과 동시에 초기화하는 것을 습관화한다

- 자료형의 기본값
	정수형 0
	실수형 0.0
	논리형 false
	문자형 ' '
	문자열 "", null

- 변수의 사용
	변수명을 이용한다
	변수안에 저장된 값이 출력된다

- 변수 선언시 주의사항
(1) 변수 이름 규칙
	영문자, 숫자, _, $만 사용 가능
	시작은 영소문자로 한다
	- 공백 포함하면 안된다, 숫자로 시작 불가능
(2) 예약어(키워드) 사용 불가
	자바에서는 이미 의미가 정해진 단어는 변수명으로 사용 불가능
	
(3) 대소문자 구분
	자바는 대소문자를 구분한다
	ex) int age;
		int AGE;
		int aGe;

(4) 의미있는 이름 사용(가독성)
	카멜케이스 사용을 권장
	변수/메소드 : 카멜케이스	totalScore
	클래스 : 파스칼케이스		PrintTest

(5) 초기화되지 않은 변수는 사용 불가능
	컴파일 에러가 발생한다
	ex) int num;
		syso(num); //에러발생

(6) 같은 범위(영역)에서 중복 선언이 불가능
	main(){
	  int num = 10;
	  int num = 20;(x)
	  num = 20;
	  syso(num); //20
	}

(7) 자료형에 맞는 값만 대입
	int num = 3.14f (x)
	int num2 = (int)3.14f (o) 강제 형변환 => 자료형을 값 앞에 ()로 같이 명시한다
	double num1 = 10; (o) 자동형변환

+) 이스케이프 문자
	문자열(String "")이나 문자(char ' ')안에서 특수한 기능을 표현할 때 사용하며 
	\(백슬래쉬)로 시작한다

	\n	줄바꿈(newline)
	\t	탭(tab)
	\'	작은따옴표 표현
	\"	큰따옴표 표현
	\\	백슬래쉬
	\r	캐리지리턴
	\b	백스페이스
	\f	폼피드

1. 입력 메소드
1) 입력 클래스
	자료형 변수명 = 값;
	String data = "문자열 변수";

	Scanner sc = new Scanner(System.in);
	이줄을 작성하면 빨간 밑줄이 뜬다 => 마우스로 빨간 밑줄있는 부분에 갖다대면 import ~
	ctrl + shift + o 	import 자동완성 단축키

2) 입력 메소드
	Scanner 클래스 안에 만들어져 있다

	sc.next();	입력받은 값을 String 타입으로 돌려준다
			엔터를 임시 저장공간에서 소모하지 않는다
			따라서 next() 사용 뒤에 nextLine() 쓰게 되면 입력받지 않고 엔터를 소모하는 역할을 하게 된다

	sc.nextLine();	입력받은 값을 String 타입으로 돌려준다
			띄어쓰기를 포함한 한 줄 전체를 입력받는다
			엔터를 임시 저장공간에서 소모한다(버퍼비우기)


	마지막에 반드시 sc.close(); 버퍼 닫아줘야한다
